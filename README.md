# ElevatedSolutions - Marcus AI Coach

Consciousness coaching powered by OpenAI for the ElevatedLives Inner Net platform.

## Setup

1. **Replace API Key**: In `script.js` line 72, replace `YOUR_OPENAI_API_KEY_HERE` with your OpenAI API key
2. **Deploy**: Push to GitHub and connect to Cloudflare Pages
3. **Configure**: Add `OPENAI_API_KEY` as an environment variable in Cloudflare

## Files

- `index.html` - Main coaching interface
- `styles.css` - ElevatedLives brand styling
- `script.js` - OpenAI integration and chat logic
- `README.md` - This file

## Features

- Marcus as your evolution guide
- Personalized consciousness coaching
- Inner Net philosophy and language
- Full conversation history
- Mobile responsive design
